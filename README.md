# Synchro Games Compass Coordinates
A simple mod to display the player coordinates when a compass is in it's inventory.

### Instalation

This mod is only compatible with fabric loaders

### Compatibility

Minecraft 1.21.4

### Requirements

Java 21
Fabric >=0.16.9
